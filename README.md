# To-Do List Project

This is a simple To-Do List application built using Django. It allows users to:

- Add new tasks.
- View a list of tasks.
- Update and delete tasks.

## Features

- Add tasks using a form.
- Edit existing tasks.
- Delete tasks.
- View tasks in a list.

## Technologies Used

- Python 3
- Django 4.x
- HTML/CSS
- SQLite (default database for Django)
